MLS Throw-in Origin-Destination Map

Click an origin zone (highlighted band along each touchline) to see where throws from that zone tend to land. The color scale changes when you select a different metric, and you can see the specific values for any tile by hovering over it. The data is based on all MLS throw-ins from 2013 through 2025.
